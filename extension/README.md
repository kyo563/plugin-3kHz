# YouTube 参加型ゲーム参加者マネージャー（Chrome拡張）

YouTubeライブチャット上の参加希望コメントを検知し、参加者管理と次回・次々回の候補表示を行う拡張です。

## 機能

- チャット文言（複数パターン）を検知して参加者を自動登録
- 各参加者に `joinNo`（参加番号）と `count`（対局参加回数）を保持
- 参加者一覧で `+1` / `-1` / `削除` 操作
- `count` が少ない順（同数時は `joinNo` が若い順）で次回3名、次々回3名を自動選出
- `配信リセット` ボタンで配信単位の状態を初期化
- データと設定をローカル保存（リロードしても保持）
- 背景をグリーンバックに切替可能
- フォントを選択可能（デフォルト/ゴシック/明朝/等幅）

## 使い方

1. Chromeで `chrome://extensions` を開く
2. 右上の「デベロッパーモード」をON
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. この `extension` フォルダを選択
5. YouTube配信ページを開く（またはリロード）

## 運用フロー（麻雀参加型の例）

1. 視聴者がチャットに参加希望コメントを投稿
2. 右上の小窓に参加者が追加される
3. 「次回」に表示された3名が次局候補
4. 対局完了後、実際に打った3名の `+1` を押す
5. 必要に応じて `-1` や `削除` で手動調整
6. 次局・次々局候補が自動再計算される（画面の再読み込み不要）

## 設定（右側パネル）

- **フォント**: 表示フォントを変更
- **背景をグリーンバック化**: クロマキー合成向け背景に変更
- **検知ワード**: 改行・カンマ区切りで複数指定
  - 例: `参加希望`, `参加きぼう`, `参加したい`

## 注意点

- 保存先はブラウザのローカルストレージ（`chrome.storage.local`）です
- 同一ユーザーの重複登録は同一配信内で防止します
- 検知ワードが空の場合は初期ワードに戻ります
