# YouTube 参加型ゲーム参加者マネージャー（Chrome拡張）

YouTubeライブチャット上の「参加希望」を検知し、参加者の管理と次回・次々回の候補表示を行う簡易拡張です。

## 機能

- チャット文言 `参加希望` を検知して参加者を自動登録
- 各参加者に `joinNo`（参加番号）と `count`（対局参加回数）を保持
- `+1` ボタンで対局実施メンバーの回数を加算
- `count` が少ない順（同数時は `joinNo` が若い順）で次回3名、次々回3名を自動選出
- `配信リセット` ボタンで配信単位の状態を初期化

## 使い方

1. Chromeで `chrome://extensions` を開く
2. 右上の「デベロッパーモード」をON
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. この `extension` フォルダを選択
5. YouTube配信ページを開く（またはリロード）

## 運用フロー（麻雀参加型の例）

1. 視聴者がチャットに `参加希望` を投稿
2. 右上の小窓に参加者が追加される
3. 「次回」に表示された3名が次局候補
4. 対局完了後、実際に打った3名の `+1` を押す
5. 次局・次々局候補が公平ルールで再計算される

## 注意点

- データはメモリ保持のみです（タブ再読み込みで消えます）
- 同一ユーザーの重複登録は同一配信内で防止します
- キーワードは現時点で `参加希望` 固定です
