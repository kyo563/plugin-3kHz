# plugin-3kHz

配信者向けに、YouTubeライブのコメントから麻雀参加希望者を管理する **Chrome拡張** です。

## 対象ユースケース

- 配信者が、コメント欄の `参加希望` を基点に参加者を管理する
- 次局・次々局の候補を公平ルール（参加回数と参加順）で確認する
- 対局が終わるごとに `+1` で参加回数を更新する

## 現在の制約

- データ保持はメモリのみ（タブ再読み込みや配信リセットで消去）
- 参加検知キーワードは `参加希望` 固定（カスタマイズ未対応）

## 導入手順（Chrome拡張）

1. このリポジトリを取得
   ```bash
   git clone <REPOSITORY_URL>
   cd plugin-3kHz
   ```
2. Chromeで `chrome://extensions` を開く
3. 右上の「デベロッパーモード」を ON
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. `extension` フォルダを選択
6. YouTube配信ページを開く（またはリロード）
